1.3.4 相等运算符

不同类型的值用相等操作符比较后的结果

类型(x) 类型(y) 结果
null undefined true
undefined null true
数字 字符串 x == toNumber(y)
字符串 数字 toNumber(x) == y
布尔值 任何类型 toNumber(x) == y
任何类型 布尔值 x == toNumber(y)
字符串或数字 对象 x == toPrimitive(y)
对象 字符串或数字 toPrimitive(x) == y

如果 x 和 y 是相同类型，JavaScript 会比较它们的值或对象值。其他没有列在这个表格中的情况都会返回false。
